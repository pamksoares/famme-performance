COMO RODAR LOCALMENTE
# Backend
cd apps/web
cp .env.example .env  # preencher variáveis
npx prisma db push
yarn dev  # http://localhost:3000

# Mobile (simulador ou dispositivo físico)
cd apps/mobile
yarn start  # Expo Go ou expo run:android

COMO GERAR APK
cd apps/mobile
EXPO_TOKEN=<token> npx eas-cli build --platform android --profile apk --non-interactive

IMPORTANTE — PATCH EXPO-MODULES-CORE
Há um bug no expo-modules-core@1.12.26 com Gradle 8.8 (from components.release). O fix está em patches/expo-modules-core+1.12.26.patch e é aplicado automaticamente via postinstall (root package.json). Não remova este patch ou os builds Android vão falhar.
